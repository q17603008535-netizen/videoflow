## 一、核心公式

```
晒过程 = 火车节骨架 + 细节讲解 + 钩子埋设 + 情绪曲线
```

缺任何一项，内容不成立：
- 无讲解 → 流水账，3秒划走
- 无钩子 → 情绪平坦，中途流失
- 无情绪曲线 → 刺激单一，用户麻木
- 无骨架 → 结构散乱，完播率低

---

## 二、晒过程本质

**为什么晒过程是内容平台的终点：**
1. **眼见为实是最大营销利器**：让用户看到手艺/服务/产品的生产过程，建立信任
2. **时长=流量**：8分钟完播30% = 240秒，远超20秒完播100%
3. **过程不同质化**：知识内容500条说尽，每次实际操作都独特
4. **用户来消磨时间**：等地铁5分钟、上厕所，不是来省时间的

**数据验证：**
- 天元邓纲（钓鱼）3300万粉丝，单条8-9分钟
- 千万级粉丝博主多数是晒过程博主
- 教知识博主极少超千万粉丝

## 三、火车节模型

**结构：**
```
开始 → 中间1 → 中间2 → 中间3 → 结果
```

**错误做法（流水账）：**
```
西红柿炒鸡蛋：
准备食材 → 炒鸡蛋 → 炒西红柿 → 一起翻炒 → 完成
（每节只有动作，无细节，无讲解）
```

**正确做法（有肉的骨架）：**
```
西红柿炒鸡蛋：
准备食材：西红柿去皮技巧（开水烫一下就撕下来）
选葱：选什么葱、不选什么葱，原因
炒鸡蛋：加什么调料能让鸡蛋特别嫩
炒西红柿：火候、注意事项
放糖：放两勺的原因
翻炒：火候控制
完成展示
```

**规则：**
- 每个节点必须有「这是什么/为什么这样做」的讲解
- 反面案例：切肉不说话只展示动作 → 观众看不懂，3秒划走
- 正面案例：永辉超市卖肉大哥，每刀讲解部位和技法 → 100万粉丝

## 四、情绪曲线原理

**核心：** 人类情绪不会长时间保持平稳。同一刺激持续超过3秒，情绪开始下降。

**反例：**
- 一首歌"动次打次"重复180分钟 → 无法忍受
- 美女对镜头笑60秒不变 → 从"真好看"到"划走"

**正例：**
- 美女亮相→走来→对你笑（3个节点）→ 点赞率高10倍

**规律：**
- 每3-5秒一个情绪波动点
- 曲线必须有低点，没有低点高点失去意义
- 苹果只有第一口最甜，情绪刺激也要保鲜

**情绪曲线模板（供参考，不必照抄）：**
```
开篇建立期待↑ → 第一节平铺→ → 中间遇到困难↘ → 转折意外↑ → 继续推进→ → 结尾爆点↑↑↑
```

## 五、9类钩子

钩子的作用：制造期待感，让用户继续看下去。

| 类型 | 定义 | 触发方式示例 |
|---|---|---|
| 荷尔蒙 | 美女帅哥、身材、颜值 | 「神秘嘉宾即将出现」 |
| 盲盒 | 不知道结果是什么 | 「闭眼买菜，买到啥做啥」 |
| 验证解密 | 能不能成功？结果如何？ | 「这个到底能不能做到？」 |
| 怀旧 | 童年回忆、经典场景 | 「这个你小时候一定见过」 |
| 惊喜 | 超预期的结果 | 「结果出乎所有人意料」 |
| 金钱 | 花小钱办大事、超值 | 「100块买一周菜，能做到吗？」 |
| 整蛊 | 捉弄、反套路 | 「老板不知道我们在测试他」 |
| 意外 | 计划外的转折 | 「突然发现了一个问题」 |
| 送温暖 | 感动、帮助他人 | 「她看到结果哭了」 |

**钩子埋设原则：**
- 开篇建立期待（hookskill 负责）
- 中间2-3个钩子维持兴趣（shai-guocheng 负责）
- 结尾产生"砰"的爆点

**不埋钩子的代价：** 情绪平坦，用户在中间段大量流失，完播率崩塌。

## 六、四大方向定义

### 方向1：做测评
- **核心：** 用实际结果说话，验证产品效果
- **正确做法：** 去取快递现场开箱 / 极限测试（打玻璃、泡水）
- **错误做法：** 坐在桌前看视频介绍 / 「这个产品一定不行」（全网烂大街）

### 方向2：接挑战
- **核心：** 完成高难度任务，时间/成本制造期待
- **示例：** 24000块拼图60小时 / 100元全屋家具布置 / 3000元装修阳台
- **钩子来源：** 倒计时 + 困难程度 + 「到底能不能完成」

### 方向3：带体验
- **核心：** 体验别人的生活/感受，角色代入
- **示例：** 男友体验分娩阵痛仪1级→10级 / 体验特殊职业
- **钩子来源：** 感受落差 + 「他们会怎么反应」

### 方向4：帮改造
- **核心：** 改造他人形象/空间，前后对比
- **示例：** 街头随机找人改造穿搭 / 帮粉丝装修 / 形象设计
- **钩子来源：** 起点越惨、终点越好 + 「最后会变成什么样」

## 七、禁止行为

| 禁止 | 原因 |
|---|---|
| 只有动作无讲解 | 流水账，观众不知道在看什么 |
| 情绪全程平坦 | 刺激疲劳，中途流失 |
| 无钩子 | 没有继续看下去的理由 |
| 结果前置不留悬念 | 开头就揭晓结果，后面没人看 |
| 视频过短（<2分钟） | 完播绝对秒数太低，流量权重低 |
| 知识讲解替代过程 | 破坏晒过程的核心—眼见为实 |

## 八、案例库（钩子设计参考）

**案例1：粉色车送网红（汽车广告，3分钟）**
- 钩子1（开始）：员工洗车特别仔细 + 粉色车 → 今晚有情况
- 钩子2（节点1）：要送给小姐姐 → 长什么样？
- 钩子3（节点2）：是网红 → 认识吗？
- 钩子4（节点3）：一直不露脸 → 持续期待
- 爆点（结尾）：揭晓是前员工
- 广告自然融入车辆介绍，不突兀

**案例2：98寸电视搬运（电视广告）**
- 钩子1：能不能进电梯？
- 钩子2：混凝土打不打得碎？
- 钩子3：进电梯严丝合缝 → 惊喜
- 爆点：画质展示，产品价值自然呈现

**案例3：菜市场买菜（金钱挑战方向）**
- 开场：「100块钱买一周菜，能做到吗？」
- 钩子1：「听说这家特别贵，能砍价吗？」
- 钩子2：「老板说今天进了稀有食材」
- 钩子3：「最后一共花了95块，够7天吗？」
- 结尾：展示所有食材 + 菜单

**手机维修案例（带讲解的正确示范）：**
```
开场：「iPhone X进水不开机，尽量修好面容」
过程讲解：「现在电流短路，是进水常见问题」
         「主板没有腐蚀迹象」
         「用松香大法检测」
         「问题出在这颗电容」
结尾：「现在开机了，资料还在，喜欢留下双击」
```
→ 有讲解 = 高完播 + 变现（修手机生意）
→ 无讲解 = 只看到拆手机 = 3秒划走
