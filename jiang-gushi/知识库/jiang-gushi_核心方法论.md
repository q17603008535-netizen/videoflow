## 一、营销故事的底层公理

**公理1：故事必须能卖货**
所有营销故事必须与最终变现挂钩。好人好事能火但不赚钱，选题前必须问：「这个故事能帮我卖货/招商/吸引客户吗？」不能，再精彩也不做。

**公理2：成就大小不重要，困境转机的戏剧性才重要**
小目标 + 精彩曲线 > 大目标 + 流水账。成就只是开篇定调，情绪过山车才是留住观众的核心。

**公理3：绝对不写努力**
「每天睡4小时」「工作16小时」「非常辛苦」= 无效内容，观众无感。
困境写具体的外部冲突（被欺负/被背叛/被抛弃），转机写具体的事件（贵人出现/发现商机/意外反转）。

## 二、三大故事类型与结构

### 类型1：个人成就型

**适用场景：** 创业历程、职业转型、从0到1的个人故事

**标准结构：**
```
开篇点题（交代成就）
→ 第1轮困境
→ 第1轮转机
→ 第2轮困境（递进，比第1轮更难）
→ 第2轮转机（比第1轮更爽）
→ 感悟升华（与变现挂钩）
```

**最低要求：** 至少2轮困境转机，一帆风顺的成就没人看

**判断信号：** 用户素材涉及「我自己」的经历、蜕变、创业历程

---

### 类型2：成功案例型

**适用场景：** 客户案例、学员成果、服务价值证明

**标准结构：**
```
交代客户初始状态（背景+痛点）
→ 客户困境（找你之前遇到的问题）
→ 你的价值出现（转机：你是客户的"通关密码"）
→ 递进困境（过程中的新障碍）
→ 递进转机（你的专业判断发挥作用）
→ 成功结果（具体数字）
```

**关键要求：** 
- 不能写「跟其他人一样这么做就好了」
- 必须写「根据他的情况，我发现这个方法特别适合」
- 你是客户的通关密码，突出独特价值和专业判断

**判断信号：** 用户素材涉及「我的客户/学员」的故事

---

### 类型3：平凡英雄型

**适用场景：** 餐饮/服务/零售老板，日常接触客户的小事

**标准结构：**
```
开篇还原小事（用特写镜头，细节呈现）
→ 讲述你的处理方式
→ 体现人品格局（讲究/仗义/有面儿）
```

**核心本质：** 「我虽然有能力占便宜，但我选择不占」

**题材清单：**
- 随手帮助客户解决小问题
- 坚持原则不占便宜（主动退定金）
- 仗义扶弱帮助弱势群体
- 敢做敢当主动承认错误
- 及时回怼面对不公发声
- 超预期服务主动补偿

**平凡英雄的对立面（禁止）：** 炫耀赚钱/炫耀资产/看不起客户

**判断信号：** 用户素材是生活中一件具体小事，主角展现了人品

## 三、困境题材黄金清单

### 个人成就型困境（必须从此选）

| 题材 | 说明 | 案例 |
|------|------|------|
| 众叛亲离 | 家人不支持、亲戚看不起、朋友疏远 | 老婆要离婚 |
| 恶霸欺压 | 同行恶意竞争、被打压驱赶 | 同行掀摊位赶出地铁站 |
| 奸人所害 | 被合伙人背叛、被客户欺骗、被员工坑 | 客户撤单50万 |
| 狂妄自大 | 一意孤行导致失败 | 奶茶店选错位置欠债被赶 |
| 强权干涉 | 政策变化、不可抗力 | 翻新机政策叫停（注意尺度） |
| 过河拆桥 | 成功后被抛弃、恩将仇报 | — |

### 成功案例型困境（客户的困境）

- 找你时濒临破产
- 不被家人理解/左右为难
- 客户自身的特殊处境（老公出差无法盯场）
- 缺乏专业知识迷茫无助
- 30岁转型压力

## 四、转机题材排行榜

**优先级从高到低：**

| 排名 | 题材 | 说明 | 经典案例 |
|------|------|------|---------|
| 1 | 因祸得福 | 最爽，观众最喜欢剧情反转 | 被赶出地铁站→换位置遇大老板 |
| 2 | 贵人相助 | 最好使最常用，最容易写出来就好使 | 张无忌遇怪人学九阳神功 |
| 3 | 天赋异禀 | 强调天生擅长某事，发现自己的突破点 | 河豚发现泪沟是突破点 |
| 4 | 意外宝藏/通关密码 | 发现商机漏洞、获得关键资源 | 发现布艺本子是蓝海 |
| 5 | 破釜沉舟 | 孤注一掷后成功 | 没日没夜赶工几个月 |
| 6 | 伙伴加入 | 团队力量突破 | — |

**成功案例型专用：**「你就是客户的通关密码」= 根据他的情况，我发现这个方法特别合适

**转机递进规则：** 小转机 → 更好的转机（地摊成功 → 布艺爆火 → 大老板订单）

## 五、情绪曲线模型

**标准过山车结构：**
```
开篇（平静/定调）→ 困境1（情绪下降）→ 转机1（上升）
→ 困境2（再下降，比第1次更难）→ 转机2（再上升，比第1次更爽）
→ 结局（高潮）
```

**流水账 vs 好故事：**
- ❌ 流水账：第一年10万，第二年20万，第三年50万（数学题）
- ✅ 好故事：赚钱了 → 遇危机 → 找到方法 → 又遇挫折 → 最终成功

**时间跨度处理：** 时间线可以跨越3-5年，但只讲2-3个关键转折点，每个都是困境→转机。中间的努力过程一句话带过。

## 六、特写镜头四要素

**必须写特写的时机：**
- 困境中最难的时刻
- 转机出现的关键瞬间
- 平凡英雄型的整个小事经过

**四要素（关键时刻必须包含）：**

| 要素 | 说明 | 示例 |
|------|------|------|
| 对话 | 还原真实对话，一字一句写出来，不概括 | 「张总，是我哪里做得不好吗？」 |
| 动作 | 具体动作描写 | 手都在抖/盯着屏幕5分钟/摔门而出 |
| 心理活动 | 当时的内心想法 | 「当时我就想就这么算了」 |
| 明确数字 | 不说「一些钱」，要说600块/3700元/50万本 | 沉默了10秒/连续3个月只有几十播放量 |

**禁止写在无关细节的特写：**
- ❌ 起床/穿衣/买早餐/开电脑（无关日常）
- ❌ 头像是蓝色花朵（无关细节）
- 判断标准：这个细节推动剧情了吗？制造情绪了吗？都没有→删掉

---

## 七、子类型判断规则

| 判断条件 | 子类型 |
|---------|--------|
| 素材主角是「我自己」的经历/创业/蜕变 | 个人成就型 |
| 素材主角是「我的客户/学员」，我是解决方案 | 成功案例型 |
| 素材是一件具体小事，展现人品/格局 | 平凡英雄型 |
| 无法判断 | 输出两个子类型选项，让用户选，不猜测 |

---

## 八、三类型内容节奏建议

不要只做一种类型，三种交替效果最好：
- 个人成就型（1-2个月1条，需要大素材）
- 成功案例型（持续可做，有客户就有素材）
- 平凡英雄型（最高频，每天都在发生，餐饮老板最适合）
