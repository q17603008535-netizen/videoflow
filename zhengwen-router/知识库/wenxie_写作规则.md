# 文案写作规则
**来源：** `/Users/mac/Desktop/方法论提纯/文案写作.md`

---

## 核心公式

```
好文案 = 特写镜头 = 画面感 + 力量感
```

---

## 一、基础原则

**写客观事实，不写感受**

说话人有感受，听者没感受。说话人说事实，听者才有感受。

```
❌ "我努力创业，小有成就"
✅ "我开了个彩票店，三个月赚了107万"
```

```
❌ 全篇："我很惨、我可怜、我非常惨、我要努力"
✅ 应该写：具体发生了什么事、现场细节是什么
```

**自检标准：村口老太太能听懂吗？**
听不懂 → 改。能听懂 → 合格。

---

## 二、画面感：两种方法

### 方法1：类比法
用熟悉事物类比陌生概念。

```
❌ "三个月减了25斤"
✅ "三个月减了25斤，相当于饮水机的一桶水"

❌ "工地上有个特大的工程车"
✅ "工地上有个车，前面那俩轮子都跟我差不多高"
```

类比必须是画面，不是感受：
```
❌ "今天星星真亮，像我心底里的希望"（不是画面）
✅ "今天星星真亮，像森林里的萤火虫"（是画面）
```

### 方法2：描述法
描述带来感受的具体行为，不说感受本身。

```
❌ "今天的风很大"
✅ "我站在那儿不用动，风能吹着我走。大衣上的帽子直接扣到了我的脑袋上"

❌ "我因为生气做了一件很作死的事"
✅ "我转头从梳妆台抄起她的口红，开盖，拧出口红，大拇指用力一摁。折了的口红掉在了地上"
```

## 三、文字使用规则

**多用：动词、名词、量词**
**少用：形容词**（如用，只用描述颜色和形状的）
**不用：成语**

```
❌ "侃侃而谈"（成语，无画面）
✅ "吐沫星子都喷我脸上"（动词+名词，有画面）

✅ 颜色/形状形容词可以用：
"一双修长白皙的双腿踩着高跟鞋，踏在老旧的石板路上，哒哒哒作响"
（动词：踩、踏、响 / 名词：双腿、高跟鞋、石板路 / 量词：一双）
```

## 四、力量感：三种方法

### 方法1：方言词汇
方言天然有力量感，普通话表达不了的情感用方言补。

```
❌ "我在后面追赶他，他在前面逃跑"
✅ "我在后面一顿撵他，他在前面嗷嗷一顿撩，鞋我都给他撵掉了"
```

常用高力量感词汇：撵、撩、块、旋、鸡头白脸、破马张飞、五脊六瘦

### 方法2：强弱对比
通过对比制造情绪曲线。

```
❌ "我赚了十万"
✅ "我出生在一个山沟里，打拼十年，成了上市公司CEO"

❌ "我失业了"
✅ "我失业了，在餐厅里吃饭的时候，真羡慕那些端盘子的服务员"
```

对比维度：数量对比 / 时间对比 / 经历对比 / 身份对比

### 方法3：程度副词
加强语气，让感受更有冲击。

```
❌ "我没喝水" → ✅ "我连口水都没喝"
❌ "我钱少" → ✅ "我钱特别少"
❌ "我伤心" → ✅ "我真是太伤心了"
```

## 六、高频错误

| 错误类型 | 错误示例 | 正确方向 |
|---|---|---|
| 写感受不写事实 | "我很惨" | 描述具体惨的场景 |
| 用成语 | "侃侃而谈" | 换成具体动作描述 |
| 类比用感受 | "像我心底的希望" | 换成可视化的画面 |
| 开篇好但后续全是感受 | 好开头+全程抒情 | 接上具体事件细节 |
| 只有骨架没有细节 | "我失业了没钱了" | 描述失业的具体场景 |
