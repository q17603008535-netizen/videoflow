# 公理与诊断规则

---

## 一、核心公理（不可违背）

**公理 1：钩子由情绪决定形态，不由模板决定**
钩子是为情绪服务的，什么情绪最强烈就用什么形态。
→ 检验：这个钩子形态是不是最能戳中这个情绪的？

**公理 2：钩子描述情境，不讲结论**
「内容没有转化不是流量问题」= 结论，不是钩子。
「发了 50 条视频，播放十几万，0 个咨询」= 情境，是钩子。
→ 检验：用户听完第一句，是点头觉得「有道理」，还是停下来觉得「这说的是我」？

**公理 3：反转是钩子的核心驱动力**
让人继续看的原因不是「这条内容有价值」，而是「我想知道他接下来说什么」。
→ 每个钩子必须有一个让人想听下去的悬念或反转。

**公理 4：口语化是钩子的基础要求**
钩子是说出来的，不是读出来的。
→ 生成后默读一遍：说话时不会这么说的词，全部替换。

**公理 5：人群属性必须来自用户确认**
不能自行假设年龄、收入、职业。
→ 每一条人群属性都要用户明确或从内容中推导出来。

## 二、钩子4要素框架（固定）

```
[身份锚定] + [具体处境/数字] + [反常识反转] + [价值承诺]
```

### 各部分说明

**身份锚定**
- 作用：让对的人在 1 秒内知道「在叫我」
- 要求：具体身份词，不用泛词
- ✅ 「做实体生意的老板」「25岁去相亲的女生」
- ❌ 「大家」「很多人」「想赚钱的朋友」

**具体处境/数字**
- 作用：触发「这说的就是我现在的状态」
- 要求：必须有具体数字或可感知的处境描述
- ✅ 「发了 50 条视频，播放量十几万，一个咨询都没来」
- ❌ 「发了很多内容，没什么效果」

**反常识反转**
- 作用：制造悬念，让人放弃划走
- 要求：打破目标客户最常见的错误认知
- ✅ 「不是流量不够，是少做了这一步」
- ❌ 「你需要优化你的内容」（这是废话，不是反转）

**价值承诺**
- 作用：给出看完的具体理由
- 要求：说清楚看完能得到什么，不用「干货满满」「建议收藏」
- ✅ 「今天用我帮中医诊所做过的案例，把这一步拆给你看」
- ❌ 「今天分享一个很有用的方法」

## 三、情绪→钩子形态映射

| 情绪核心 | 钩子形态 | 开头特征 |
|---------|---------|---------|
| 恐惧 | 直接戳痛点 | "最怕..." "千万不能..." |
| 嫉妒/攀比 | 制造反差对比 | "凭什么他..." "为什么别人..." |
| 身份认同 | 锁人开场 | "做___的停一下..." "___的人注意..." |
| 好奇 | 设问悬念 | "你知道为什么..." "你有没有发现..." |
| 共鸣 | 说经历引共情 | "有没有人和我一样..." "我也是..." |
| 逆反 | 反常识反转 | "你以为___？其实..." "不是___，是___" |
| 虚荣 | 身份加持 | "戴上这个..." "用这个..." |

## 四、自检标准（四项，逐条检查）

| 标准 | 对应规则 | 判断方式 |
|---|---|---|
| 前 3 秒有具体处境或数字 | 前 3 秒跳出率是冷启动生死线 | 第一句话能不能在 3 秒内说完 |
| 目标客户能在 3 秒内对号入座 | 账号标签精准度决定推流质量 | 有没有明确的身份词 |
| 有反转悬念 | 2026 复看率权重排第一 | 听完第一句后是否想知道「接下来是什么」 |
| 人群属性来源清晰 | 推导逻辑必须有依据 | 用户确认或内容中明确 |

任意一项不达标 → 标注问题，不输出该条钩子。
全部不达标 → 说明原因，重新推导，不输出空结果。

---

## 五、推导流程（强制顺序）

```
Phase 1：人群属性推导（问清楚：身份+年龄+场景+产品）
    ↓
Phase 2：人性弱点推导（根据人群+选题选取弱点）
    ↓
Phase 3：情绪痛点提炼（从弱点得出三层痛点）
    ↓
Phase 4：生成钩子（根据情绪选形态，套4要素）
    ↓
Phase 5：自检（4项标准逐条检查）
    ↓
Phase 6：输出
```

不得跳过 Phase 1 直接进入 Phase 2。
不得在人群属性未确认时生成钩子。

---

## 六、弱点组合规则

1. **核心弱点最多2个**：超过2个必须收敛到主要矛盾
2. **可能存在多种组合时**：输出多种让用户选择，不强行定论
3. **组合要有明确核心情绪**：每个组合对应一个清晰的情绪方向
4. **人之常情不是弱点**：合理回报诉求、正常竞争心理不算弱点

## 七、表达规范（必须遵守）

### 禁止使用的句式

| 禁止句式 | 原因 |
|----------|------|
| "在成都混，你还没来过..." | 已经被用烂 |
| "你是不是那个..." | 套路化 |
| "不是...而是..." | 太书面 |
| "今天告诉你..." | 说教感 |
| "家人们..." | 讨好感 |
| "99%的人都不知道..." | 夸张 |
| "学会这个，月入百万..." | 标题党 |

### 表达技巧

1. **用具体场景替代抽象概念**
   - ❌ "你需要优化内容"
   - ✅ "你发了50条没人看"

2. **用疑问句替代陈述句**
   - ❌ "我来告诉你一个方法"
   - ✅ "你有没有发现..."

3. **用挑衅替代说教**
   - ❌ "你要这样做"
   - ✅ "你是不是一直做错了"

4. **用身份替代人群**
   - ❌ "很多人都有这个问题"
   - ✅ "开火锅店的你，有没有..."

### 网感原则

1. **短句**：每句不超过15字
2. **直接**：去掉"其实"、"实际上"、"一般来说"
3. **具体**：数字要真，身份要准
4. **挑衅但不说教**：让用户好奇，不是让用户觉得你厉害

### 自检清单

生成钩子后检查：
- [ ] 口语化：默读一遍，说不出来就改
- [ ] 无老套句式：对照禁止句式检查
- [ ] 具体数字：有没有真实数字
- [ ] 身份精准：能不能3秒内对号入座
