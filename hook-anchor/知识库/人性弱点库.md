# 人性弱点库

钩子推导的核心依据。每次推导钩子必须从这里取，不得脱离逻辑链自行编造。

---

## 底层人性弱点清单（含准确定义）

### 傲慢
- **定义**：不能丢人、不能比别人差、必须要有面子
- **边界**：是"我不能输"，不是"我要赢"
- **区分**：vs 嫉妒 - 傲慢是"我要比你强"，嫉妒是"你凭什么比我好"
- **表现**：攀比、身份认同、要被羡慕、不愿承认错误

### 嫉妒
- **定义**：对别人拥有的好东西感到眼红、不服气
- **边界**：是"凭什么他有我没有"，不是"我也想要"
- **区分**：vs 贪婪 - 嫉妒是"他超过我了"，贪婪是"我想要更多"
- **表现**：逆袭心理、打脸、证明自己、不服输

### 恐惧
- **定义**：怕失去现有状态、怕被淘汰、怕错过机会
- **边界**：是"怕变差/被落下"，不是"想要更好"
- **区分**：vs 焦虑 - 恐惧有具体对象（被淘汰/失去），焦虑是模糊的落后感
- **表现**：FOMO、限时、最后机会、趋势警告、怕落后

### 焦虑
- **定义**：怕落后、怕被时代抛弃、怕不确定性
- **边界**：是弥散的"怕被抛下"，没有具体对象
- **区分**：vs 恐惧 - 焦虑更模糊，恐惧有具体威胁
- **表现**：趋势分析、阶层警告、对比、中产焦虑

### 虚荣
- **定义**：想要外在符号带来的认同感，要让人看得起
- **边界**：是"我要让人觉得我厉害"，不是"我要真的变厉害"
- **区分**：vs 傲慢 - 虚荣需要别人看到，傲慢是内心确认
- **表现**：奢侈品、同款、晒、身份加持

### 懒惰
- **定义**：想不劳而获、走捷径、不想付出努力
- **边界**：是"我不想干活"，不是"我想高效干活"
- **区分**：vs 贪婪 - 懒惰是"我不想付出"，贪婪是"我想得到更多"
- **表现**：3天学会、速成、躺赚、捷径、副业神话

### 贪婪
- **定义**：对超出所需的过多欲望，自私且过度
- **边界**："人之常情的回报诉求"不是贪婪，"超出需要的贪得无厌"才是
- **区分**：vs 勤奋 - 想要合理回报是正常的，贪婪是"既要又要还要"
- **表现**：all-in、押上身家、借钱投资、想一夜暴富

### 孤独
- **定义**：怕被群体排斥、需要有归属、需要被认同
- **边界**：是"我不能被落下"，不是"我想被喜欢"
- **区分**：vs 虚荣 - 孤独是被排斥感，虚荣是被仰慕感
- **表现**：社交货币、圈子、入群、怕单身

## 常见误区（必须避免）

| 误区 | 正确理解 |
|------|---------|
| "想赚钱"=贪婪？ | 否。想赚合理利润是正常的。"想一夜暴富、押上身家"才是贪婪 |
| "不想白干"=贪婪？ | 否。任何人做事都希望有回报，这是人之常情 |
| "羡慕别人"=嫉妒？ | 否。欣赏是"我也想这样"，嫉妒是"他凭什么比我好" |
| "怕落后"=焦虑？ | 可能。焦虑是模糊的怕落后，恐惧是有具体对象的怕 |

## 人性弱点组合逻辑

同一选题可能涉及多个弱点叠加：

| 组合 | 情绪痛点 |
|------|---------|
| 傲慢+嫉妒 | 不能在同学会上丢人、凭什么他行我不行 |
| 虚荣+傲慢 | 买奢侈品的目的是身份认同、被看得起 |
| 恐惧+焦虑 | 怕被AI取代、怕落后于时代 |
| 嫉妒+懒惰 | 眼红别人成功但不想付出 |
| 孤独+虚荣 | 需要圈子认同、晒给别人看 |

## 人群×弱点交叉推导

### 爱美女性（25-50岁）
- 弱点：傲慢、虚荣、嫉妒
- 场景：相亲、职场、社交、婚姻
- 痛点：不能丑、不能老、不能被比下去、被看得起

### 实体店老板
- 弱点：傲慢、嫉妒、恐惧
- 场景：生意、引流、竞争
- 痛点：不能亏钱、不能没客人、不能被同行比下去

### 职场人群
- 弱点：焦虑、嫉妒、恐惧
- 场景：晋升、加薪、裁员
- 痛点：不能落后、不能被看不起、怕被淘汰

### 年轻人（学生/新人）
- 弱点：虚荣、焦虑、嫉妒
- 场景：社交、职场、消费
- 痛点：不能被嘲笑、不能落伍、眼红别人成功

### 中老年
- 弱点：恐惧、孤独、傲慢
- 场景：健康、养老、家庭
- 痛点：不能生病、不能孤独、要有尊严

## 推导原则（强制）

1. **从选题推导弱点**，不是从弱点套选题
2. **弱点必须有定义支撑**，不能望文生义
3. **区分人之常情和人性弱点**，合理回报不是贪婪
4. **弱点组合要合理**，不是越多越好
5. **人群属性决定弱点表现**，同一弱点不同人群表现不同
