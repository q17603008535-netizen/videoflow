# 人设八维提取框架

---

## 8个维度定义与提取线索

| 维度 | 定义 | 录音中的识别线索 |
|---|---|---|
| 身份 | 职业/角色标签 | 「我是做___的」「我在___」「我的工作是」 |
| 地域 | 所在城市或地区 | 「在___」「___本地」「我们这边」 |
| 年龄 | 年龄段或出生年代 | 「我___年」「90后」「做了___年」 |
| 行业 | 所在垂直领域 | 「___行业」「做___生意」「我们这个赛道」 |
| 性格 | 明显的个性特征 | 说话的语气、处事态度、自我描述 |
| 行为 | 标志性的日常行为 | 「我每天___」「我习惯___」「我一般都」 |
| 收入/成就 | 可量化的成果背书 | 金额、规模、数量、时间跨度 |
| 外部背书 | 第三方可信度来源 | 粉丝量、获奖、媒体报道、合作品牌、资质认证 |

---

## 提取原则

1. **只提取录音中明确出现的信息**，不推测、不补充
2. **模糊信息标注「待补充」**，不用推测值填充
3. **同一维度有多个信息时全部保留**，用分号分隔
4. **数字要具体**：不写「很多」，写「30个」；不写「很久」，写「5年」

---

## 性格特征识别补充说明

性格特征不由用户自述决定，而是从说话方式和态度中提炼：

| 录音中的表现 | 对应性格标签 |
|---|---|
| 说话直接、不绕弯 | 直率、不废话 |
| 频繁提到失败或遗憾 | 反思型、真实感强 |
| 多次强调某个原则或底线 | 有主见、有立场 |
| 语气幽默、自嘲 | 接地气、有亲和力 |
| 频繁用数字和逻辑 | 理性、数据导向 |
