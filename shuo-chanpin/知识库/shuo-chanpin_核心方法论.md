# 说产品核心方法论知识库

## 一、底层认知

**说产品是最难火但最好写的类型。** 用户来抖音是娱乐，不是看广告。突破策略：把一个点打透，降低制作成本，靠量取胜。

**产品三分类：**
- 价格低 → 价格型脚本
- 功能强 → 功能型脚本（四段式）
- 颜值高 → 颜值型脚本

**选主打点原则：** 开篇只打一个点留住人，其他功能放中后段。不知道哪个卖点好？出5条视频分别测试。

---

## 二、结构类型判断规则

| 产品主打点 | 使用结构 | 视频时长 |
|-----------|---------|---------|
| 功能/痛点解决 | 四段式（场景难题→产品推荐→使用效果→引导下单） | 30-60秒 |
| 价格震撼 | 价格对比型（三段式：价格震撼→产品卖点→引导下单） | 15-30秒 |
| 颜值/礼物 | 视觉展示型（两段式：产品特点+区别→适用人群+购买理由） | 30-45秒 |

---

## 三、功能型脚本（四段式）详解

### 第一段：场景难题（10秒内）
**核心：说用户痛点场景，不是产品功能**

❌ 错误：直接说产品功能
✅ 正确：描述真实生活痛点场景，用细节和画面感，让用户产生"这说的就是我"的共鸣

模板：
```
描述用户正在经历的某个具体麻烦场景
用细节强化画面感（时间节点/具体动作/反应）
```

实战案例参考：
- 洗衣液："结了婚后给老婆洗内裤，那不是常有的事吗？"
- 洗地机："如果没有洗地机，我都不知道如何开启二胎生活"
- 下水道疏通剂："眼瞅着快过年了，下水道下不去水，还呼呼往上反味，头疼吧？"

### 第二段：产品推荐（10秒内）
**快速切入，只说1个核心功能，口语化**

模板选用：
- "我跟你说怎么办？[产品名称]，它[核心功能]，就你[使用场景]"
- "还好有[产品]拯救了我的[痛点]，这个不仅能[功能1]，还能[功能2]"
- "我老婆一直让我用的都是这款[产品]，它里面含有[成分]，[痛点1][痛点2]通通都能解决"

### 第三段：使用效果（20秒内）
**从四个维度说效果（选2-3个）：**

1. 效果好：解决程度、速度、彻底性
2. 使用方便：操作简单、省时省力
3. 安全性：不伤害、无副作用
4. 附加价值：额外好处、意外惊喜

模板：
- "用它[操作]过的[对象][效果]，[具体数据/对比]"
- "用完之后不用自己[麻烦操作]，直接[简单操作]就可以"
- "它不会[常见担心]，也不会[副作用]，可以放心[使用]"
- "而且它还能[额外功能]，这样就不用担心[延伸问题]"

### 第四段：引导下单（5秒内）
**必须有行动指令 + 制造紧迫感**

模板选用：
- "记住了啊，[核心卖点总结]，趁着[时间节点]快递还没停，有需要的赶紧[下单动词]"
- "[价格]谁都能买，[数量/规格]，[行动指令]"
- "这个[价格]就能[获得价值]，真的很有必要入手一个，[行动指令]"

## 四、价格型脚本（价格对比型）详解

**核心：开篇就说价格震撼感，节奏快，靠量取胜**

### 第一段：价格震撼（3秒）
金句模板：
- "卷疯了！[价格][数量/规格][产品]"
- "[价格]你能想到[夸张数量]吗？[具体数量]，还是[附加价值]"
- "我们家很少[消费行为]，但这个[价格]一大包[产品]，实在是没人拒绝得了"

备用开篇金句：
- "今天这一波就跟不要钱似的"
- "[价格]你敢信？"
- "这年头[价格]还能买到这个？"

### 第二段：产品卖点（10秒）
从5个维度选2-3个快速罗列：
1. 大小/数量：多大、多少个、够几个人用
2. 颜色/口味：有几种选择
3. 材质/工艺：什么材料
4. 功能/特点：独特之处
5. 产地/品牌

模板："这个[产品]是[材质/工艺]，[功能特点]，有[颜色/口味]，[使用场景]，特别[形容词]"

### 第三段：引导下单（5秒）
- "[价格][规格]，赶紧选/挑吧"
- "[价格]趁着[时间节点]，赶紧整"
- "[价格]就能[获得价值]，点一下吧"

---

## 五、颜值型脚本（视觉展示型）详解

**核心：场景布置+文案质感，适合礼物属性产品**

### 第一段：产品最大特点+区别（20秒）
模板：
- "我第一次看到这么[程度]的[产品]，它比[常见同类]更[差异点]，更像是[升维比喻]"
- "设计非常[形容词]，[特征1]，[特征2]，[特征3]"
- "[操作方式]，[视觉效果]，[感受描述]，会非常[形容词]"

### 第二段：适用人群+购买理由（10秒）
模板：
- "如果你的[对象]喜欢[爱好]，对[追求]有一定的追求，这会是一个非常有[特质]的[场景]礼物"
- "这个礼物送给[人群]真的是没有[负面]，这种东西你不送他，他可能一辈子见都没见过，用了之后[改变]"
- "这个时候你就成功了"

## 六、结构类型自动判断逻辑

根据 BOOMskill 输出的选题方向和爆款元素，按以下优先级判断：

1. 选题中出现价格数字（X块X/X.X元/X元X件）→ **价格对比型**
2. 选题中出现"痛点/难题/解决/终于"类关键词 → **四段式**
3. 选题中出现"颜值/礼物/高端/精致/送人"类关键词 → **视觉展示型**
4. 无明显信号时，默认 **四段式**（适用范围最广）

---

## 七、老郭人设硬性规则

**必须有（全文至少各一个）：**
- 具体数字（≥1个）：价格、时间、数量、规格
- 反常识陈述（≥1条）：打破用户预期的表达
- 短句节奏：多用10字以内短句，制造口播感

**绝对禁止：**
- 自问自答式开头（"你是不是也有这样的烦恼？"）
- 模糊主语：很多人/大部分人
- 书面语：然而/因此/此外/综上
- 排比堆砌：A、B、C、D四大优势

**禁止词列表：**
相信自己 / 加油 / 坚持 / 努力 / 你可以的 / 很多人 / 大部分人 / 可能 / 也许 / 干货 / 绝了 / 首先其次最后 / 建议收藏

## 八、导演版标注规范

导演版在纯净版基础上增加：
- `【情绪】`：激动/平静/困惑/惊喜/认真等
- `【镜头】`：特写/全景/展示产品/展示效果/跟镜等
- `【动作】`：拿出产品/倒入/展示包装/指向等
- `【节奏】`：加速/停顿/放慢等
- `（括号内为导演备注）`：提示拍摄注意事项
